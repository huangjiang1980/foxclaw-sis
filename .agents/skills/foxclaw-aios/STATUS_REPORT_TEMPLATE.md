# System Status Report Template | 系统状态报告模板

## Usage | 使用方法

Use this template to check and report FoxClaw SIS system health.

**Activation Command:**
```
超智系统状态检测
FoxClaw SIS status check
```

---

## Report Template | 报告模板

```
🦊 FoxClaw SIS 状态检测报告
*检测时间: [YYYY-MM-DD HH:MM]*

|------|------|------|

| 🔄 进化层 | [百分比] | [状态描述] |
| 🔌 插件生态 | [百分比] | [状态描述] |
| 🛠️ 执行层 | [百分比] | [状态描述] |
| ❤️ 生命支持 | [百分比] | [状态描述] |
```

## Status Indicators | 状态指示器

| Indicator | Meaning | 含义 |
|-----------|---------|------|
| 🟢 | Healthy / Normal | 健康/正常 |
| 🟡 | Warning / Attention | 警告/需注意 |
| 🔴 | Critical / Error | 严重/错误 |
