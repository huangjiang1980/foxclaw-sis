# 🦊 威科夫股票分析技能

**版本：** 2.0.2 | **更新：** 2026-03-12

---

## 📖 简介

基于威科夫（Richard Wyckoff）经典量价理论，专为中国A股市场设计的股票分析技能。

> "威科夫是核心，大资金动向是参考辅助"

---

## 🎯 核心功能

| 功能 | 描述 |
|------|------|
| 多周期分析 | 月线定方向 → 周线找机会 → 日线精确入场 |
| 量价分析 | 最近3天量价关系，判断主力意图 |
| 形态识别 | Spring/UT/SOS/LPS 等关键信号识别 |
| 双层止损 | 生命线止损 + 灵活止损体系 |
| 自动盯盘 | 实时监控，触发条件第一时间通知 |
